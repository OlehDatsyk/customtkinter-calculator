# CustomTkinter Calculator

A sleek, modern GUI calculator built with Python and the `customtkinter` library. This application features a responsive grid system, dark mode styling, and basic arithmetic execution.

## 🚀 Features

* **Modern UI:** Sleek, modern design using CustomTkinter widgets with a native dark theme.
* **Intuitive Layout:** Responsive button grid with custom spanning (the large "0" button).
* **Core Math Operations:** Supports addition, subtraction, multiplication, division, and parenthesis handling.
* **Error Handling:** Safe execution using a `try-except` block that catches syntax or evaluation errors gracefully without crashing the app.

## 🛠️ Prerequisites

Before running this project, you will need to have Python installed on your machine. You will also need the `customtkinter` library.
